# Descriptive and Diagnostic Analysis of Cybersecurity Incidents

# 1. Overview of Cybersecurity Incidents

To understand the overall cybersecurity posture of the organization, descriptive analytics was first applied to calculate the total number of security incidents recorded during the study period. The analysis revealed multiple incidents across different departments, indicating that cybersecurity threats are not isolated to a single function but affect the organization as a whole.
